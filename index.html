TRACK

Frontend Engineering

Detail: Front-end web development,also known as client-side development is the practice of producing HTML,CSS and Javascript for a website or web Application so that a user can see and interact with them directly.

Language: HTML, CSS, Javascript, Node.js etc

Backend Engineering

Detail: The Backend of a website is a combination of technology and programming that power a website. Its consists of three parts that a user never sees: a server, an application, and a database.

Language: PHP, Ruby, Python, Java etc

Cloud Engineering

Detail: Cloud computing is the delivery of computing resources as a service. Moving to the cloud basically means that the resources are owned and managed by a third-party provider, instead of the end-user.

Language: NET, Golang, python, PHP etc
